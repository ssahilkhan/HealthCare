# HealthWeb

A modern health and wellness web application built with React, featuring Radix UI components and Supabase backend integration for managing health-related data and user interactions.

## Progress

![Progress](https://progress-bar.dev/60)

## Tech Stack

- **React 19** (Vite) - Frontend framework
- **Radix UI** - Accessible component primitives (Accordion, Alert Dialog, Avatar, Checkbox, Dialog, Dropdown Menu, Label, Slider, Tabs, Toast)
- **Tailwind CSS** + 	ailwindcss-animate - Utility-first styling
- **Supabase** - Backend-as-a-Service (PostgreSQL database, authentication)
- **Framer Motion** - Animations
- **Lucide React** - Icon library
- **React Router DOM** - Client-side routing
- **class-variance-authority** + **clsx** + **tailwind-merge** - Component styling utilities

## Future Scope
